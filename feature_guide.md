# Durio Feature Guide - Primary Overview

This document provides a high-level overview of the entire Durio application. For detailed, button-by-button explanations of each module, refer to the individual deep-context files located in the `docs/features/` directory.

## Application Architecture

Durio is built around a centralized layout with a persistent Header and Navigation Sidebar, allowing users to seamlessly transition between core productivity and utility modules. 

### Core Modules Summary

1. **[To-Do (Tasks)](file:///d:/myprojects/durio/docs/features/todo.md):** 
   A comprehensive task management system featuring a Kanban board, sub-checklists, recurring schedules, and visual streak tracking to encourage productivity.

2. **[Notes](file:///d:/myprojects/durio/docs/features/notes.md):** 
   A rich-text document editor organized via a color-coded folder shelf, supporting deep linking to tasks and calendar events.

3. **[Calendar](file:///d:/myprojects/durio/docs/features/calendar.md):** 
   A unified chronological dashboard displaying standard events, specialized occasions (which automatically calculate age or anniversaries), and local weather data.

4. **[Calculator](file:///d:/myprojects/durio/docs/features/calculator.md):** 
   A massive suite of mathematical utilities including Scientific, Graphing, Matrix algebra, Complex polynomial operations, and Statistics engines.

5. **[Settings](file:///d:/myprojects/durio/docs/features/settings.md):** 
   The global preference center where users can manage their profile, disable unneeded modules to declutter their sidebar, and toggle "Fancy Mode" to optimize device performance.

## Layout Elements (Header & Sidebar)
The application's frame consists of a top Header (housing global sync, theme switching, and an interactive clock) and a left-side Navigation Menu (housing routing links and dynamic visual decorations). Detailed breakdowns of these layout elements can be found in **[Layout Elements Guide](file:///d:/myprojects/durio/docs/features/layout.md)**.
